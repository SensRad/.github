# Sensrad
*Vision beyond the visible*

Sensrad develops 4D imaging radar hardware and perception software for autonomous systems operating in demanding real-world environments. Our technology delivers reliable detection and situational awareness across all weather and lighting conditions — rain, snow, fog, dust, and darkness.

---

## Products

| Product | Description |
|---|---|
| [Hugin D1](https://www.sensrad.com/hugin-radar-d1) | High-resolution dynamic 4D imaging radar for autonomous vehicles, robotics, and mobile platforms |
| [Oden Drive](https://www.sensrad.com/oden-drive) | Radar-based perception stack for autonomous vehicles, including odometry, free-space and drivable surface estimation, object tracking, and classification |
| [Oden Eye](https://www.sensrad.com/oden-eye) | Radar-based detection system for UAVs and aerial applications |
